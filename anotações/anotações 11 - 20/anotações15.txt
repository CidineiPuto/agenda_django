No main do site, colocamos o content, que é algo que sempre estará lá.

Para injetar os contatos, iremos puxar do models a classe do contacts, e puxar
tudo de lá.


from contact.models import Contact


def index(request):
    contacts = Contact.objects.all()

Agora iremos criar o context.

    context = {
        "contacts": contacts,
    }
    return render(
        request,
        "contact/index.html",
        context,
    )

Depois iremos retornar o render.

  {% for contact in contacts %}
    {{ contact.first_name }} <br>
  {% endfor %}

Depois iremos fazer um for, o importante é que já está injetado, mas depois terminamos
isto, os organizando.
