IRemos colocar a header como partial.
Normalmente quando o arquivo é partials, ele possui um "_" antes do nome.
O head, ele terá que ser acompanhado junto com a chamada do static;

    {% load static %}
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="{% static 'global/css/style.css' %}">

E assim, irá funcionar normalmente.

    context = {
        "contacts": contacts,
        "site_title": "Contatos - ",
    }

Agora ficará sendo mostrado o título do site.

    site_title = f"{single_contact.first_name} {single_contact.last_name} - "

    context = {
        "contact": single_contact,
        "site_title": site_title,
    }

Agora irá mostrar o nome da pessoa.
