O site não irá ficar no nosso computador, afinal, iremos mandar ele para o sever.
E vamos ter que mudar coisas no settings, mas pode ser gerados transtornos,
já que vamos ter uma coisa aqui, e outra no server.

Vamos criar um arquivo que não é monitorado pelo git, chamado local_settings.py.
Logo vamos tentar importar tudo que está nesse arquivo, pois qualquer coisa colocado
no local_settings vai ir para settings.

    try:
        from project.local_settings import *
    except ImportError:
        ...

Lá serve para colocar as coisas quando for para o servidor.

    DEBUG = False
    ALLOWED_HOSTS = ["*"]

Esse ALLOWED_HOSTS permite que qualquer site que tentar acessar vai dar sucesso.
Esse local host, irá sobescrever as outras configurações.
Mas também pode ser usado o dotenv do django.
