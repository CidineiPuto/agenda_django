Agora vamos colocar algo para exibir a imagem.

  {% if contact.picture %}
    <p>
      <img src="{{ contact.picture.url }}" alt="{{ contact.first_name }} {{ contact.last_name }}">
   </p>
  {% endif %}

Porém pode levantar um erro, caso não tenha nenhuma imagem para exibir.
Por isso o "if" é usado.
Agora, para poder colocar a imagem vamos usar "picture";

        fields = (
            ...
            "picture",
        )

Isso é dentro do forms, dentro da class meta.
Porém, o django coloca uma imagem padrão, mas vamos exibir a imagem.

    class ContactForm(forms.ModelForm):
        picture = forms.ImageField(
            widget=forms.FileInput(
                attrs={
                    "accept": "image/*",
                }
            )
        )

Iremos reescrever o widget para isso ocorrer.
Porém, também vamos mostrar a imagem dentro do local.
Dentro do for do create.html;

      {% if field.name == 'picture' and field.value.url %}
        <div class="form-group">
          <img src="{{ field.value.url }}" alt="">
        </div>
      {% endif %}

Quer saber se dentro do field.name possui o campo "picture" e se nesse picture,
possui um valor url.

Colocamos isso fora da form-content, e criamos uma div para a img.
Agora, precisamos fazer com que caso a pessoa troque de foto a imagem seja alterada.

No request tem dois parâmetros, o primeiro sendo a data e o segundo é o request.FILES
que é os arquivos enviados;

    if request.method == "POST":
        form = ContactForm(request.POST, request.FILES)

Todo lugar que tiver colocando post no forms, será colocado files.

Lembra que no forms tem algo chamado (enctype="multipart/form-data") em que vamos precisar
dele para isso.
Assim a imagem dele será trocada.
