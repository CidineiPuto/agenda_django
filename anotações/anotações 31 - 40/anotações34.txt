Nós precisamos fazer alguma coisa com ContactForm(request.POST) no "contact_forms.py";

    if request.method == "POST":
        form = ContactForm(data=request.POST)
        context = {"form": form}

Vamos usar o form.is_valid()

        if form.is_valid():
            print("form é válido")

O form do django, e o form.is_valid faz com que o form não seja salvo.
Então podemos fazer o seguinte;

        if form.is_valid():
            form.save()

Se o form for válido, ele será salvo.
Podemos pegar o contato para fazer mais algo com ele;

            contact = form.save(commit=False)

Com o commit false, ele fará com que o contact ainda não seja salvo.

        if form.is_valid():
            contact = form.save(commit=False)
            contact.show = False
            contact.save()

O contato será salvo após ele for dado como show=False.

Precisamos mandar os contatos para outra view aonde os dados serão editados.
