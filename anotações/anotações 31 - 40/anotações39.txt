do django, no user.
Vamos começar recriando o meta da class;

from django.contrib.auth.models import User

Usamos essa importação para pegar o model.

    class RegisterForm(UserCreationForm):
        class meta:
            model: User
            fields = (
                "first_name",
                "last_name",
                "email",
                "username",
                "password1", <- Senha
                "password2", <- Confirmação de senha.
            )

O django não deixa existir dois usuários com o mesmo nome, mas podemos fazer a mesma coisa,
porém, com o email.

    def clean_email(self):
        email = self.cleaned_data.get("email")

        if User.objects.filter(email=email).exists():
            self.add_error(
                "email", ValidationError("Email already exist", code="invalid")
            )
        return email

Estamos pegando isso através do clean_email, aonde podemos levantar um erro com if.
O "exists()" retornar um bool de false ou true.
