Agora que é possível enviar o form, lá na action por enquanto vamos colocar a url;

    <form 
    action="{% url "contact:create" %}"
    method="POST"
    enctype="multipart/form-data"
  >

Garante que vai enviar essa url no action do form.
No nosso name temos o seguinte nome;

    <input type="text" name="first_name" maxlength="255" id="id_first_name">

name e id tem que ser único nas páginas. O name é requerido, pois através dele pega a chave
dentro do request.
O post pode vir em dict, então para pegar ele, pode ser feito o seguinte;

    search = request.POST.get('first_name')

Se for digitado na parte algo como "luiz otávio" isso irá pegar este nome.
Para ver o method usado, pode fazer o seguinte;

    print(request.method)

Quando o form estiver sendo lido, é possível perceber que o method será get, mas quando
for enviado, o method será post.
Na view, ou tem get ou post.
O post não pode ser passado tão facilmente, então podemos fazer o seguinte;

    if request.method == "POST":
        print()
        print(request.method)
        print(request.POST.get("first_name"))
        print(request.POST.get("last_name"))
        print()

Normalmente nessa parte é para salvar o form.
